# 🎓 want-to-graduate


> "졸업까지 남은 길, 클릭 한 번으로 확인하세요."


**want-to-graduate**는 Java Swing으로 만든 데스크톱 앱으로,  
사용자가 들은 수업만 선택하면 **졸업 가능 여부를 계산**해줍니다.

## 🧠 프로젝트 개요

이 프로젝트는 경기대학교 컴퓨터공학부 학생들이 스스로 졸업 요건을 점검할 수 있도록 설계된 **졸업 요건 검증 시스템**입니다.  

사용자는 “수강한 과목”을 선택하기만 하면 총 이수 학점 · 전공필수 과목 이수 여부 · 부족 학점을 자동으로 계산하여 시각적으로 보여줍니다.
